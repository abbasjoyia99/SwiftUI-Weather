# SwiftUI-Weather
**Overview**
This is a sample iOS app that demonstrates Weather application Screens UI with SwiftUI.
The app incorporates:

SwiftUI
ZStacks
VStacks
HStacks
Images and Text
Color Gradients
Models
State
Binding
DarkView
LightView
